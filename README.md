Currency Wallet & Exchange System
Микросервисная система для управления мультивалютными кошельками, обмена валют и мониторинга крупных транзакций.

Архитектура
Система состоит из трёх независимых микросервисов:

gw-currency-wallet — основное приложение для управления кошельком пользователя

gw-exchanger — сервис получения и хранения курсов валют

gw-notification — сервис обработки крупных транзакций


Технологический стек
Бэкенд
Язык: Go (Golang)

Фреймворк: Gin для REST API

Аутентификация: JWT (JSON Web Tokens)

Межсервисная коммуникация: gRPC

Асинхронная обработка: Kafka

Базы данных: PostgreSQL (основные данные), MongoDB (история транзакций)


Функциональность
gw-currency-wallet
Регистрация и авторизация пользователей

Управление мультивалютным кошельком (USD, RUB, EUR)

Пополнение и вывод средств

Обмен валют с кэшированием курсов

RESTful API с JWT-аутентификацией

gw-exchanger
Хранение и предоставление курсов валют

gRPC API для доступа к курсам

Поддержка USD, RUB, EUR

Абстракция хранилища для гибкости БД

gw-notification
Асинхронная обработка крупных транзакций (от 30 000)

Сохранение истории операций

Надёжная обработка до 1000 сообщений/сек

Graceful shutdown поддержка
